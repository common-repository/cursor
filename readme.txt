== Description ==

This is a plugin which replaces the standard cursor of your OS to an Cursor of your choise. Just install and replace the .cur files. Much fun :) .



== Installation ==

Install and open your FTP-Client. Go to the directory /wp-content/plugins/cursor/img and replace the current .cur files and rename them to the names of the current standard files
